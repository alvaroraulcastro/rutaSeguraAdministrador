# 🚐 RutaSegura — Panel de Administración Web

Este repositorio contiene la aplicación **Web Admin** del sistema **RutaSegura**, una plataforma integral para la gestión de transporte particular de personas.

## 🚀 Tecnologías

Este proyecto está construido con:

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **UI Framework:** [Ant Design (antd)](https://ant.design/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Despliegue:** [Vercel](https://vercel.com/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)

## 📋 Funcionalidades Principales

- **Dashboard:** Monitoreo en tiempo real de rutas activas y KPIs de operación.
- **Gestión de Transportistas:** Registro y administración de conductores y vehículos.
- **Gestión de Pasajeros:** Control de usuarios, domicilios, destinos y contactos de emergencia.
- **Gestión de Rutas:** Configuración inteligente de recorridos de ida y vuelta.
- **Log de Notificaciones:** Seguimiento detallado de alertas enviadas a pasajeros (Push, SMS, WhatsApp).

## 🛠️ Instalación y Desarrollo

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 📁 Estructura del Proyecto

- `src/app`: Directorio principal con las páginas y rutas (App Router).
- `src/components`: Componentes reutilizables de la interfaz.
- `docs/`: Documentación del proyecto, incluyendo el [Plan de Desarrollo](./docs/plan-desarrollo-app-transporte.md) y los [Wireframes](./docs/wireframes-web-admin.md).
